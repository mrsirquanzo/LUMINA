## 🔧 Fix Plan

### PHASE 1: Critical UI Fixes (Priority 1)

#### 1.1 Fix Badge Design
**Issue:** Badge too big, says "Generated by [Agent]"

**Solution:**
- Remove "Generated by" text
- Just show: `[Icon] [Agent Name] • [time ago]`
- Reduce badge size: `px-2 py-0.5` instead of `px-2 py-1`
- Smaller font: `text-xs` instead of `text-sm`
- Compact layout: single line, no wrapping

**Files to Modify:**
- `src/components/DynamicTile.tsx` (lines ~240-250)

**Implementation:**
```typescript
// Before:
<div className="flex items-center gap-1.5 px-2 py-1 rounded-full text-xs font-medium ...">
  <span>{agentInfo.icon}</span>
  <span className="hidden sm:inline">Generated by {agentInfo.name}</span>
  <span className="sm:hidden">{agentInfo.name.split(' ')[0]}</span>
  <span className="text-gray-500">•</span>
  <span className="text-gray-500">{timeAgo}</span>
</div>

// After:
<div className="flex items-center gap-1 px-1.5 py-0.5 rounded-full text-xs font-medium ...">
  <span className="text-sm">{agentInfo.icon}</span>
  <span>{agentInfo.name}</span>
  <span className="text-gray-500 mx-1">•</span>
  <span className="text-gray-500">{timeAgo}</span>
</div>
```

```typescript
export function matchQueryToScenario(query: string): string | undefined {
  const normalizedQuery = query.toLowerCase().trim();
  
  // HER2 matching (add this)
  if (normalizedQuery.includes('her2') || normalizedQuery.includes('her-2')) {
    return 'her2-analysis';  // Need to create this scenario
  }
  
  // KRAS G12C matching
  if (normalizedQuery.includes('kras') && normalizedQuery.includes('g12c')) {
    return 'kras-g12c';
  }
  
  // ADC/Enhertu/Trodelvy matching
  if (normalizedQuery.includes('enhertu') || normalizedQuery.includes('trodelvy') || 
      normalizedQuery.includes('padcev') || normalizedQuery.includes('adc') ||
      (normalizedQuery.includes('compare') && normalizedQuery.includes('competitive'))) {
    return 'competitive-analysis';
  }
  
  // ... other matches
  
  // Don't default - return undefined if no match
  return undefined;
}
```

## 🎨 Design Specifications

### Badge Design (Fixed)
```
[Icon] Agent Name • 2m ago
- Size: text-xs, px-1.5 py-0.5
- Layout: Single line, no wrapping
- Colors: Agent-specific with proper contrast
```

### Tile Content Structure (Rich)
```
Tile Header:
- Title
- Badge (compact)
- Actions (pin, delete)

Tile Body:
- Tabs (if applicable)
- Charts/Visualizations
- Metrics Cards
- Detailed Content
- Sources/References
```

### Text Legibility Standards
- Dark text on light: `text-gray-900`
- Light text on dark: `text-white`
- Secondary text: `text-gray-600` or `text-gray-400`
- Minimum contrast ratio: 4.5:1
