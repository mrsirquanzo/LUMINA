## Overview

The Multi-Agent Collaboration System orchestrates three specialized AI agents—Clinical Data Analyst, Patent Expert, and Financial Analyst—to work together on complex biotech analysis scenarios. The agents don't just run in parallel; they actively collaborate by asking each other questions, sharing context, and building on each other's insights.

**Key Innovation:** Real-time inter-agent communication where agents can dynamically request information from their peers, creating a collaborative analysis that's more comprehensive than any single agent could provide.

## What It Can Do

### 🤝 M&A Due Diligence
**Scenario:** Should we acquire GeneTech for $800M?

The system analyzes:
- **Clinical:** Phase 2 efficacy (35% ORR), safety profile, competitive benchmarking
- **Patent:** IP strength, FTO status, 20-year exclusivity assessment
- **Financial:** Burn rate analysis, DCF valuation ($680-750M), milestone-based deal structure

**Output:** Comprehensive acquisition recommendation with integrated risk assessment across all dimensions.

---

### 💼 Investment Decisions
**Scenario:** Should we invest $25M in Series B?

The system evaluates:
- **Clinical:** Phase 1 data quality, regulatory pathway assessment
- **Patent:** Patent portfolio strength, competitive positioning
- **Financial:** Runway analysis, capital requirements, exit scenarios

**Output:** Investment thesis with cross-domain risk factors and deal terms recommendation.

---

### 📊 Competitive Intelligence
**Scenario:** Compare competitive positioning of three CAR-T therapies

The system compares:
- **Clinical:** Efficacy benchmarking, safety profiles, patient populations
- **Patent:** IP landscapes, blocking patents, white space analysis
- **Financial:** Market sizing, revenue potential, commercialization costs

**Output:** Competitive analysis matrix with strategic positioning recommendations.

---

### 🔬 Licensing Deals
**Scenario:** Should we license this mRNA delivery platform for $50M upfront?

The system analyzes:
- **Clinical:** Technology validation, clinical applications
- **Patent:** IP strength, FTO for key indications, patent valuation
- **Financial:** Deal structure evaluation, royalty terms, risk-adjusted NPV

**Output:** Go/no-go recommendation with optimized deal terms.

## Key Features

**Real-Time Collaboration** ⚡
Agents ask each other questions during analysis, creating a collaborative dialogue visible in real-time via Server-Sent Events (SSE).

**Cost Transparency** 💰
See estimated cost before running analysis. Choose between Fast mode (quick, lower cost) and Thorough mode (comprehensive, higher value).

**Demo Mode** 🎬
Experience multi-agent collaboration with pre-recorded scenarios—no API costs. See realistic agent dialogue for M&A, licensing, and investment use cases.

**Export Capabilities** 📥
Download complete synthesis as Markdown or copy to clipboard. Share strategic recommendations with stakeholders.

**Document Upload** 📄
Upload your own documents (PDFs, Excel, images) and ask custom questions for live multi-agent analysis.

## Technical Architecture

<details>
<summary><strong>Click to expand technical details</strong></summary>

**Orchestration Engine**
- Sequential and parallel execution modes
- Dynamic question parsing and routing
- Context accumulation across agent interactions
- Real-time SSE streaming to frontend

**Inter-Agent Communication**
- Regex-based question extraction: `[ASK_CLINICAL: "question"]`
- Targeted agent consultation with full context
- Q&A history tracking for consistency
- Answer integration into original responses

**AI Models**
- Claude Sonnet 4 for all agents
- 4K token context per agent call
- 2K token responses for targeted questions
- Adaptive token budgets based on mode

**Frontend**
- React with SSE client
- Real-time progress visualization
- Agent dialogue streaming
- Live cost tracking

**Cost Optimization**
- Token-based estimation before execution
- Mode selection based on use case
- Parallel vs sequential tradeoffs
- Demo mode for showcasing (zero cost)
