## Overview

AI-powered analyst for SEC filings, financial models, and biotech valuations. Upload 10-K/10-Q documents, Excel models, or investor relations URLs to get instant analysis of burn rate, runway, and investment risks. Built with **Gemini 2.0 Flash** featuring massive context windows (1M+ tokens) for processing entire filings.

**Data Sources:** 📊 SEC EDGAR API • 💹 Alpha Vantage • 💼 Financial Data APIs

## Technical Details

<details>
<summary><strong>Click to expand technical implementation details</strong></summary>

**AI Model**
- Gemini 2.0 Flash (gemini-2.0-flash-exp)
- Vision API for financial charts and graphs
- 1M+ token context window - processes entire 10-K filings

**Document Processing**
- PDF parsing via pdf-parse library
- Excel/CSV processing with xlsx and papaparse
- PowerPoint extraction for investor presentations
- Base64 encoding for image analysis

**API Integrations**
- SEC EDGAR API (real-time filing retrieval)
- Alpha Vantage (stock data and market metrics)
- Financial Data APIs (valuation multiples, benchmarking)

**Architecture**
- Next.js 14 with App Router
- TypeScript for type safety
- Server-side API routes for document processing
- Client-side React components for interactive UI

**Security**
- Session-based authentication
- Environment variable configuration
- No document storage (privacy-preserving)
- All processing happens in real-time

</details>
