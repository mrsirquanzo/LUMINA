## Overview

The Patent Intelligence Agent is an AI-powered assistant that analyzes patent documents, competitive IP landscapes, and freedom-to-operate questions. Upload patent PDFs, provide Google Patents URLs, or paste patent text to get instant analysis of claims, prior art, competitive positioning, and strategic recommendations.

**Built with Perplexity** featuring vision capabilities for analyzing patent drawings and figures.

## Example Questions

Here are specific questions the agent can answer when you upload relevant patent documents:

### Claims Analysis
- "What are the key independent claims in this patent and how broad is their scope?"
- "Identify the most commercially valuable claims in this patent family"
- "What limitations are present in Claim 1 that could be designed around?"
- "Compare the claim scope across US, EP, and CN patents in this family"

### Freedom-to-Operate
- "Does this patent block our antibody-drug conjugate approach?"
- "Identify potential blocking patents for CAR-T cell therapy with IL-15 costimulation"
- "What are the infringement risks for our manufacturing process?"
- "Assess freedom-to-operate for this combination therapy indication"

### Competitive Intelligence
- "What patents does Company X have in the gene therapy space?"
- "Analyze Moderna's mRNA delivery platform patent strategy"
- "Track filing trends for CRISPR base editing over the past 3 years"
- "Which companies dominate the ADC linker technology landscape?"

### Strategic Planning
- "What white space exists in the bispecific antibody patent landscape?"
- "Identify gaps in our patent coverage for this target"
- "Recommend defensive publication topics to block competitor filings"
- "Assess patent strength and validity risks for this portfolio"

## Supported Formats

**Documents**
- PDF (granted patents, patent applications, office actions)
- Plain text (patent claims, abstract, description)

**Images**
- Patent drawings and figures (analyzed with AI vision)
- Flowcharts and diagrams from patent specifications

**Web Content**
- Google Patents URLs
- USPTO, EPO, WIPO patent database links
- IP news and analysis articles

**Limits**: Up to 5 patents per query, 10MB max file size

---

## Key Features

**Citation Tracking** ✅
Every insight references the specific patent, claim number, or section it came from. Citations appear inline as `[Source: US1234567A, Claim 1]`.

**Multi-Patent Comparison**
Upload multiple patents from a family or competitive portfolio and compare claims, scope, and strategic positioning side-by-side.

**Vision-Enabled**
Claude's vision API analyzes patent drawings, flowcharts, and chemical structures directly from images or PDF figures.

**Patent Family Awareness**
The agent understands relationships between related patents (continuations, divisionals, foreign counterparts) when you provide multiple related documents.

## Technical Details

**AI Model**
- Perplexity
- Vision API for patent drawing analysis
- 4K token context window per query

**Document Processing**
- PDF parsing for patent documents
- Text extraction from USPTO/EPO formats
- HTML extraction for web-based patent databases
- Base64 encoding for patent drawing analysis

**Architecture**
- Next.js 14 with App Router
- TypeScript for type safety
- Server-side API routes for document processing
- Client-side React components for interactive UI

**Security**
- Session-based authentication
- Environment variable configuration
- No patent storage (confidentiality-preserving)
- All processing happens in real-time
