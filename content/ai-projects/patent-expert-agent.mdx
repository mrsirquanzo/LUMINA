## Overview

AI-powered patent analyst for landscapes, freedom-to-operate (FTO) searches, and IP strategy. Upload patent PDFs or provide patent numbers to get instant analysis of claims, prior art, competitive portfolios, and patentability assessments. Built with **Perplexity Sonar Pro** featuring real-time USPTO/EPO database access.

**Data Sources:** 🇺🇸 USPTO • 🔍 Google Patents • 🇪🇺 EPO (European Patent Office)

## Technical Details

<details>
<summary><strong>Click to expand technical implementation details</strong></summary>

**AI Model**
- Perplexity Sonar Pro (llama-3.1-sonar-large-128k-online)
- Real-time web search integration for live patent data
- 128K token context window

**Document Processing**
- PDF parsing via pdf-parse library
- Vision API for patent drawings and chemical structures
- HTML extraction for USPTO/EPO patent pages

**API Integrations**
- USPTO Public Search (11M+ US patents)
- Google Patents (100M+ global documents)
- EPO Register (European patent database)

**Architecture**
- Next.js 14 with App Router
- TypeScript for type safety
- Server-side API routes for document processing
- Real-time search capabilities via Perplexity

**Security**
- Session-based authentication
- Environment variable configuration
- No document storage (privacy-preserving)
- All processing happens in real-time

</details>
