## Overview

The Clinical Data Analyst Agent is an AI-powered assistant that analyzes clinical trial reports, study data, and competitive intelligence documents. Upload PDFs, Excel files, images, or provide URLs to get instant analysis of efficacy endpoints, safety profiles, biomarker trends, and market intelligence.

**Built with Claude Sonnet 4** featuring vision capabilities for analyzing charts, graphs, and figures from scientific publications.

## Supported Formats

**Documents**
- PDF (clinical trial reports, publications, regulatory documents)
- Excel/CSV (study data, endpoint tables, patient demographics)
- Plain text files

**Images**
- PNG, JPG, WebP (charts, graphs, Kaplan-Meier curves, waterfall plots)
- AI vision automatically extracts data from figures

**Web Content**
- Any URL (ClinicalTrials.gov, company press releases, conference abstracts)
- Automatic content extraction and analysis

**Limits**: Up to 5 files per query, 10MB max file size

---

## Key Features

**Citation Tracking** ✅
Every insight references its source document, so you know exactly where each finding came from. Citations appear inline as `[Source: filename.pdf]`.

**Multi-Document Analysis**
Upload multiple trials and compare them side-by-side. The agent synthesizes information across documents to provide comprehensive analysis.

**Vision-Enabled**
Claude's vision API analyzes charts, graphs, and figures directly from images or PDFs—no manual data entry required.

**Document History**
All uploaded files are tracked with timestamps. Easily re-analyze previous documents or compare with new data.

### Gosset.ai Integration Highlights

The Clinical Data Analyst features cutting-edge pharmaceutical intelligence through Gosset.ai:

- **Phase Transition Rates (PTRs)**: Historical success rates by indication, modality, and phase
- **Trial Success Predictions**: Evidence-based probability estimates from verified trial outcomes
- **Trial Design Benchmarks**: Average sample sizes, endpoints, durations, and biomarker prevalence
- **Integration Mode**: Hybrid live/simulated with automatic fallback (zero configuration required)

When `GOSSET_OAUTH_TOKEN` is configured, real-time PTR data is fetched from Gosset's MCP server. Otherwise, it gracefully falls back to simulated pharmaceutical intelligence data for portfolio demonstrations.

## Technical Details

<details>
<summary><strong>Click to expand technical implementation details</strong></summary>

**AI Model**
- Claude Sonnet 4 (claude-sonnet-4-20250514)
- Vision API for image and chart analysis
- 4K token context window per query

**Document Processing**
- PDF parsing via pdf-parse library
- Excel/CSV processing with xlsx and papaparse
- HTML extraction with cheerio for URLs
- Base64 encoding for image analysis

**Architecture**
- Next.js 14 with App Router
- TypeScript for type safety
- Server-side API routes for document processing
- Client-side React components for interactive UI

**Security**
- Session-based authentication
- Environment variable configuration
- No document storage (privacy-preserving)
- All processing happens in real-time

</details>
