## Overview

AI-powered analyst for clinical trials, efficacy data, and competitive intelligence. Upload PDFs, Excel files, images, or URLs to get instant analysis with automatic citations. Built with **Claude Sonnet 4** featuring vision capabilities for analyzing charts and figures.

**Data Sources:** 🧬 Gosset.ai Pharmaceutical Intelligence • 🔬 ClinicalTrials.gov • 📚 PubMed E-utilities

> **Gosset.ai Integration:** Access Phase Transition Rates (PTRs) and trial benchmarks across 100K+ drug assets via live MCP connection. Provides evidence-based success predictions with hybrid live/simulated mode.

## Technical Details

<details>
<summary><strong>Click to expand technical implementation details</strong></summary>

**AI Model**
- Claude Sonnet 4 (claude-sonnet-4-20250514)
- Vision API for image and chart analysis
- 200K token context window

**Document Processing**
- PDF parsing via pdf-parse library
- Excel/CSV processing with xlsx and papaparse
- HTML extraction with cheerio for URLs
- Base64 encoding for image analysis

**API Integrations**
- Gosset.ai MCP server (SSE transport)
- ClinicalTrials.gov API v2
- PubMed E-utilities (NCBI)

**Architecture**
- Next.js 14 with App Router
- TypeScript for type safety
- Server-side API routes for document processing
- Client-side React components for interactive UI

**Security**
- Session-based authentication
- Environment variable configuration
- No document storage (privacy-preserving)
- All processing happens in real-time

</details>
